# NeGuLa

A basic NGL message spammer writen in python.






## Installation

#### Windows

Just download the repo zip file and double click on main.py

#### Linux
```bash
    git clone https://github.com/moyosito/negula.git
    cd negula
    pip install -r requirements.txt
    python main.py
```
